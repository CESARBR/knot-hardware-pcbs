These are the open-source hardware PCBs of the KNoT Project.

The ECAD tool used is Altium Designer versions 14.2.3 e 14.2.4.

Each PCB projec includes:
 - Components Library
 - Schematics
 - Board Layout

And the generated output files are:
 - Bill Of Materials
 - Gerber
 - NC Drill
 - PCADNetlist
 - Assembly Drawing
 - Composite Drawing
 - Pick and Place
 - Schematics(PDF)
